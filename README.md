# auto-click-wayland
Est une petite adapatation de l'application original "Clicker".

=> Lancer l'application 
=> régler les paramètres a votre guise 
=> appuyer sur F9 pour lancer l'auto-click 
  => confirmer l'écran, autoriser l'interaction à distance et le partage
=> placer la souris a l'endroit voulu
=> Pour stopper l'auto-click appuyer sur F9

Attention le lancement de l'auto-click est instantanné, pas de reglage 5sec ou autre, le delai est by-passé, je n'ai pas corrigé ce defaut.
